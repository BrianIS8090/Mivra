# План реализации системы плагинов и Export PDF

## Цель

Реализовать безопасную v1-архитектуру плагинов в Mivra и встроенный плагин `export-pdf`, который экспортирует текущий Markdown-документ в PDF.

## Ключевые решения

- Встроенный плагин называется `Export PDF`.
- Пользовательская кнопка в Toolbar называется `Export PDF`.
- Основной сценарий экспорта создаёт PDF, а не HTML.
- Генерация PDF выполняется через `pdfmake`.
- Превью показывает отдельные листы, границы листов и общее количество листов.
- Старый id `document-designer` мигрируется в `export-pdf`, чтобы не сломать существующие настройки.

## Проверки

- Unit-тесты для манифеста, store-миграции, postраничного HTML-превью и pdfmake document definition.
- `npm run test`
- `npx tsc --noEmit`
- `cargo test`
- `npm run build`
- `graphify update .`
